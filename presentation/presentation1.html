Prerequisites

  • Basic Python familiarity

What is Pandas?

  • A Python package providing fast, flexible and expressive data structures
  • Designed to make working with relational or labeled data both easy and intuitive
  • Filling out the gap of data analysis and modelling that was not as easy to do using Python before

Why should I want to use Pandas?

  • Enables you to carry out entire data analysis workflow in Python
  • Many available libraries to perform data analysis and visualization easily
  • Not having to learn another separate language, such as R
  • Python in general is a great option to perform textual analysis

Capability

  • Data import/export
  • Data cleaning
  • Data analysis
  • Data visualization
  • Fast and efficient DataFrame object for data manipulation with integrated indexing
  • Tools for reading/writing data between in-memory data structures and formats
  • Intelligent data alignment and integrated handling of missing data
  • Reshaping and pivoting of datasets
  • Intelligent slicing, fancy indexing, subsetting of large datasets
  • High performance merging and joining of datasets
  • Time series functionality

Other Popular Statistical Software

  • Open source software: R
  • Proprietary software: Mathematica, MATLAB, SASS, SPSS

Comparison between Pandas and R

  • Syntax is often similar, one can be occassionally simpler than other based on a specfic method
  • Python has main packages for data analysis tasks, but R was developed 16 years before Pandas, so it has larger ecosystem of smaller packages
  • Python can be more efficient in non-statistical areas, such as text analysis. So, learning Pandas can serve multiple purposes.
  • Both are useful and complement each other.

Data Structures in Pandas

Series and DataFrames

Series

Series is a one-dimensional labeled array capable of holding any data type (integers, strings, floating point numbers, Python objects, etc.). The axis labels are collectively referred to as the index.

Example of Series

Data on different occupations

0 Librarian
1 Software Developer
2 Engineer
3 IT Support Specialist
4 Research Data Coordinator

DataFrame

A DataFrame is a tabular data structure comprised of rows and columns, akin to a spreadsheet, database table, or R's data.frame object. It can be thought as a group of Series objects that share an index (column names).

Example of DataFrame

About Jupyter Notebook

What is a Jupyter Notebook?

Jupyter is an interactive notebook (documents) that can contain both computer code (e.g. python) and rich text elements (paragraph, equations, figures, links, etc.). Notebooks are produced and run by the Jupyter Notebook app. It makes code sharing and execution easier to perform data analysis by scientists.

Brief History

The IPython was developed in 2011 by a team of researchers led by Fernando PÃ©rez from UC Berkeley and Brian Granger from California Polytechnic State University. The idea was to incorporate detailed and understandable descriptions of raw code, be able to keep track of different versions of code that produce various figures.

The project was later renamed as Jupyter Notebook to make it compatible with other programming languages, such as Julia and R.

Why use a Jupyter Notebook?

  • Free and open source - widely used
  • Ease of sharing
  • Literate programming document - ability to incorporate rich texts with code
  • Reproducibility

Nature published an article on the benefits of using Jupyter Notebook.

Installation instructions

Jupyter Notebook can be installed easily on your local machine using Anaconda. Experienced Python users can use pip alternatively.

Detailed instruction can be found in the Jupyter Documentation

Cells in a Jupyter Notebook

    Each section of a Jupyter notebook is called a cell. A cell can typically contain either Python code or Markdown, although with Jupyter add-ons, more formats are possible.

    Cells make it easier to execute a specific portion of your code or split between code and documentation easily. It helps keep your notebook more organized.

    Cell types: Code, Markdown, Raw NBConvert, Heading.
